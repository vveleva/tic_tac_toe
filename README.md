### CLI version of the classic Tic Tac Toe game

#### How to play:

```
SYNOPSIS
      play.rb [<player>] [<player>] [<size>]

OPTIONS
      <player>
          can be ai or human
      <size>
          is optional
```

For example:
```sh
> ruby play.rb human ai
```
